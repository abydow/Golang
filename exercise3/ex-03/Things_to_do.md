## To check The contents of your folder(Go Workspace)

```bash
ls -la
```

### Or you can Do what I like the most for small folders or pojects 

```bash
tree
```
> If it doest work for you install it with a guide and check it out 

## To build 

- You can use 
```bash
go build <file_name>
```

- or if you want to execute multiple at once use 

```bash 
go build .
```

## To run the executable

use 

```bash
./<executable_name>
```
