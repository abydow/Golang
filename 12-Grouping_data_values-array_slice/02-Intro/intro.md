## Array

- A numbered sequence of Values of the same type
- Does not change in size
- [Array Reference](https://go.dev/ref/spec#Array_types)

## Slice

- Built on top of array
- Holds values of the same type
- changes in Size
- has length and a capacity
- [Slice Reference](https://go.dev/ref/spec#Slice_types)

## Map

- Key/value storage
- an unordered group of values of one type, called element type, set by unique keys of another type, called key type
- [Map Reference](https://go.dev/ref/spec#Map_types)

## Struct

- A data structure
- A composite type/ aggregate
- Allows us to collect values of different types together
- [Struct Reference](https://go.dev/ref/spec#Struct_types)
