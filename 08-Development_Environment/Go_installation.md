# GOlang Development Environment :- 

## Go installation 

   -  Visit | https://go.dev/dl/ | and doownload the version according to your OS/ requirement 

## Git-SCM installation

   -  visit | https://git-scm.com/install | and download your suitable version

## install VS Code

   -  visit | https://code.visualstudio.com/download | and install latest version

    - Then install Go extension in vscode 
    - Then upgrade Go Tools using the command palette (click on the gear Icon then click on command palette and serach for Go:Install/Update Tools. click on it and select all then press ok)
